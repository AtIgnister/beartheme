# Beartheme
## How do I install this?  
1. Download the latest release and install it as an [unpacked extension.](https://webkul.com/blog/how-to-install-the-unpacked-extension-in-chrome/)  
2. Go to the settings page of the extension and add a repo url. The URL of the official repo is https://raw.githubusercontent.com/AtIgnister/beartheme/refs/heads/main/serv/theme.json. The list of comma seperated, so you can add more than one repo.
3. Have fun!
